## Tutorial 1: First-Time Setup

**Provider Table (Main Content)**

1. **Scroll down** to see the provider table

## Tutorial 2: Basic Search

### Objective
Learn how to perform basic searches to find providers.

### Step-by-Step Instructions

#### Step 1: Locate the Search Bar

1. **Find the search bar** at the top of the page
   - It's below the stats dashboard
   - Large, prominent input field
   - Has a search icon (üîç) on the left

2. **Click inside the search bar**
   - The border will animate with a gradient effect
   - A cursor will appear

#### Step 2: Perform Your First Search

1. **Type a search term**
   - Example: Type "Centralia"
   - As you type, results update automatically
   - No need to press Enter or click a button

2. **Observe the results**
   - The table below updates in real-time
   - Only matching providers are shown
   - A results count appears above the table

3. **Try different search terms**
   - Type "Medicaid" to find Medicaid providers
   - Type "LLC" to find Limited Liability Companies
   - Type a phone number like "360" to find providers with that area code

#### Step 3: Clear Your Search

1. **Click the X button** that appears in the search bar
   - The X appears on the right side when text is entered
   - Clicking it clears the search

2. **Or delete all text** manually
   - Use Backspace or Delete key
   - Results return to showing all providers

#### Step 4: Understand Search Behavior

**What Gets Searched:**
- Business names (e.g., "Arianna Care")
- Provider names
- City names (e.g., "Centralia")
- Phone numbers (partial matches work)
- Website URLs
- Additional notes/descriptions

**Search Characteristics:**
- **Case-insensitive**: "CENTRALIA" = "centralia" = "Centralia"
- **Partial matching**: "Arian" finds "Arianna Care"
- **Real-time**: Results update as you type
- **Instant**: No page reload required

## Tutorial 3: Understanding Provider Information

### Objective
Learn to read and interpret all the information displayed for each provider.

### Step-by-Step Instructions

#### Step 1: Examine a Provider Row

1. **Find a provider row** in the table
   - Each row represents one provider
   - Rows alternate between white and light gray

2. **Hover your mouse** over a row
   - The row highlights with a gradient background
   - Makes it easier to track which row you're reading

#### Step 2: Understand Each Column

**Column 1: Area/City**
- **What it shows**: Location of the provider
- **Format**: "City, State" (e.g., "Centralia, WA")
- **Visual indicator**: Green dot (‚óè) before the city name
- **Example**: "‚óè Centralia, WA"

**Column 2: Name of Business**
- **What it shows**: Official business name
- **Format**: Full legal business name
- **Example**: "Arianna Care LLC"
- **Note**: This is the registered business name

**Column 3: Name of Provider**
- **What it shows**: Individual provider or owner name
- **Format**: Person's name or business name
- **Example**: "Arianna Care" or "John Smith"
- **Note**: May be the same as business name

**Column 4: Phone Number**
- **What it shows**: Contact phone number
- **Format**: 10-digit number (e.g., "3608074237")
- **Display options**:
  - **Clickable link**: Blue text with phone icon (üìû)
  - **Yellow badge**: "Contact via AFHC" (no direct phone)
- **Functionality**: Click to call (mobile) or copy (desktop)

**Column 5: Website**
- **What it shows**: Provider website or AFHC listing
- **Format**: URL or "View on AFHC"
- **Display**: Blue link with external link icon (‚Üó)
- **Functionality**: Opens in new tab when clicked

**Column 6: Year Started**
- **What it shows**: Year business was established
- **Format**: 4-digit year (e.g., "2020")
- **Display**: Blue badge with year number
- **Special cases**: Shows "N/A" if unknown

**Column 7: Resident Beds**
- **What it shows**: Number of resident beds available
- **Format**: Number followed by "Beds" (e.g., "6 Beds")
- **Display**: Green badge
- **Special cases**: Shows "N/A" if unknown

**Column 8: Payment Ratio**
- **What it shows**: Private Pay vs Medicaid ratio
- **Format**: Text description
- **Common values**:
  - "100% Private Pay"
  - "100% Medicaid"
  - "50/50"
  - "Not specified"
- **Display**: Purple badge

**Column 9: Others**
- **What it shows**: Additional information
- **Format**: Text description
- **May include**:
  - Address details
  - Special services
  - Additional notes
- **Display**: Plain text (may be truncated)

#### Step 3: Interpret Status Indicators

**Color-Coded Badges:**

- **Green**: Positive/active status
  - Green dot: Active provider
  - Green badge: Beds available
  
- **Blue**: Informational
  - Blue badge: Year started
  - Blue link: Clickable elements
  
- **Purple**: Payment-related
  - Purple badge: Payment ratio information
  
- **Yellow**: Warning/missing info
  - Yellow badge: "Contact via AFHC" (no direct phone)

**Icons:**

- **üìû Phone icon**: Clickable phone number
- **‚Üó External link icon**: Opens in new tab
- **‚óè Green dot**: Active provider status

## Tutorial 4: Advanced Search Techniques

### Objective
Master advanced search techniques to find specific providers efficiently.

### Step-by-Step Instructions

#### Step 1: Search by Multiple Criteria

**Combining Search Terms:**

1. **Type multiple words** in the search bar
   - Example: "Centralia Medicaid"
   - This finds providers in Centralia that accept Medicaid
   - Words don't need to be in order

2. **Try different combinations**
   - "Centralia LLC" - LLC providers in Centralia
   - "Medicaid 360" - Medicaid providers with 360 area code
   - "Care Home" - Providers with both words in name

#### Step 2: Search by Phone Number

1. **Enter phone number digits**
   - Example: Type "360807"
   - Finds all providers with phone numbers containing these digits
   - Works with partial numbers

2. **Search by area code**
   - Type "360" to find all 360 area code providers
   - Type "206" to find all 206 area code providers

3. **Search by full number**
   - Type full 10-digit number: "3608074237"
   - Finds exact match if available

#### Step 3: Search by Business Type

**Common Business Types:**

- **"LLC"**: Limited Liability Companies
- **"AFH"**: Adult Family Homes
- **"Home"**: Any business with "Home" in name
- **"Care"**: Any business with "Care" in name
- **"Manor"**: Providers with "Manor" in name
- **"House"**: Providers with "House" in name

**Try searching for:**
- "LLC" to see all Limited Liability Companies
- "AFH" to see Adult Family Home providers
- "Care LLC" to see care businesses that are LLCs

#### Step 4: Search by Payment Type

**Payment-Related Searches:**

- **"Medicaid"**: Providers accepting Medicaid
- **"Private"**: Private pay providers
- **"Both"**: Providers accepting both
- **"50/50"**: Providers with 50/50 split

#### Step 5: Narrow Down Results

**Refining Your Search:**

1. **Start broad, then narrow**
   - First: Search "Centralia" (many results)
   - Then: Search "Centralia Medicaid" (fewer results)
   - Finally: Search "Centralia Medicaid LLC" (very specific)

2. **Use specific terms**
   - Business name: "Arianna Care"
   - Provider name: "John Smith"
   - Exact phrases work best

### What You've Learned

‚úÖ Combining multiple search terms  
‚úÖ Searching by phone numbers and area codes  
‚úÖ Finding providers by business type and payment method  
‚úÖ Refining searches for specific results

### Practice Exercise

1. Find all LLC providers in Centralia
2. Find all Medicaid providers with 360 area code
3. Find a specific provider by typing part of their business name
4. Experiment with different search term combinations

## Tutorial 6: Contacting Providers

### Objective
Learn how to contact providers using the information displayed.

### Step-by-Step Instructions

#### Step 1: Identify Contact Information

1. **Find the Phone Number column**
   - Look for the phone number in the table
   - May show as clickable link or "Contact via AFHC"

2. **Check the Website column**
   - Look for website URL or "View on AFHC" link
   - Provides additional contact options

#### Step 2: Use Phone Numbers

**On Mobile Devices:**

1. **Click the phone number**
   - Phone number appears as blue link with phone icon (üìû)
   - Clicking opens your device's phone dialer
   - Number is pre-filled, ready to call

2. **Make the call**
   - Review the number
   - Tap the call button
   - Connect with the provider

**On Desktop Computers:**

1. **Copy the phone number**
   - Right-click on the phone number
   - Select "Copy" from the context menu
   - Or manually select and copy (Ctrl+C or Cmd+C)

2. **Use the copied number**
   - Paste into your phone app
   - Use with VoIP services (Skype, etc.)
   - Save to contacts

#### Step 3: Access Provider Websites

1. **Click the website link**
   - Link appears in the Website column
   - Shows URL or "View on AFHC"
   - Has external link icon (‚Üó)

2. **Website opens in new tab**
   - Doesn't navigate away from the platform
   - Can return easily
   - Multiple provider sites can be open

3. **Explore provider information**
   - View detailed provider pages
   - Find additional contact methods
   - Learn more about services

#### Step 4: Handle "Contact via AFHC" Providers

**When you see "Contact via AFHC":**

1. **Click "View on AFHC" link**
   - Opens the provider's official AFHC listing
   - May have contact form or additional info

2. **Use AFHC contact methods**
   - Fill out contact form on AFHC website
   - Use AFHC's messaging system
   - Follow AFHC's contact procedures

3. **Note the limitation**
   - No direct phone number available
   - Must use AFHC as intermediary
   - May take longer to get response
